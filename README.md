# pust

A minimal, web-based breathing exercise designed to help you relax and focus.

## Features

- **Visual Breathing Guide**: A soothing, animated “zen circle” that expands and contracts to guide your breathing rhythm.
- **Distraction-Free**: Once the timer starts, all controls disappear so you can focus entirely on your breath.
- **Screen Wake Lock**: Prevents your screen from dimming or turning off during your session.
- **Minimalist Design**: Clean, dark interface to reduce distractions and eye strain.
- **Zero Dependencies**: Built with pure HTML, CSS, and JavaScript.

## Getting Started

Simply open `index.html` in any modern web browser to start your session.
